# tetris_game
파이썬 파이게임(pygame)을 이용한 게임제작 - 3. 테트리스 게임 개조
<br><br><br>


## 사용법
n-ris 파일 그대로 다운로드 후 적당한 실행 환경에서 코드 실행
<br>
&nbsp;개조할 때 파이썬 IDLE로 했음.
<br><br><br>
